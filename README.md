# Jarvis AI Assistant 🤖

A Python-based AI assistant inspired by JARVIS with voice automation and system control features.

## Features
- Voice command support
- Browser automation
- Desktop control
- Internet tools
- Text-to-speech
- Modular architecture

## Tech Stack
- Python
- SpeechRecognition
- pyttsx3
- APIs
- JSON

## Project Structure
```bash
data/
templates/
brain.py
voice.py
main.py
```
